# Punjabi translations — fluent-speaker review checklist

**Status:** unreviewed. AI-generated draft.

These translations were generated by Claude Code on 2026-04-25 against the English source strings in `Dosely/Resources/en.lproj/Localizable.strings`. They have **not** been reviewed by a fluent Punjabi speaker. **Do not show this build to the grandparents** until every row below has been signed off.

## How to review

For each row:

1. Read the **English** column for context.
2. Read the **Punjabi** column.
3. If the Punjabi reads naturally to a 65+ Punjabi-Canadian speaker, mark the **Status** column ✅.
4. If it's stiff, formal, robotic, mistranslated, or uses words a grandparent wouldn't recognise, mark ✏️ and suggest a replacement in the **Reviewer note** column.
5. Tone target: how a younger family member would *speak* these phrases to grandma — not how a textbook would write them.

Specific things to watch for:

- **Drug names** are kept in Latin script in `drug_info_pa.json` (e.g., "Metformin") because BC pharmacies print bottles in English. Don't translate the drug names themselves.
- **Numerals** stay Western Arabic (1, 2, 3) by design. See "Numbers" in CLAUDE.md.
- **Medical terms** ("ਖ਼ੂਨ ਦਾ ਦਬਾਅ" for blood pressure, "ਸ਼ੂਗਰ" for sugar/diabetes, etc.) — confirm with a healthcare-adjacent reviewer if possible.
- **The word for "medication"** — I used "ਦਵਾਈ"; if your reviewer prefers "ਦਵਾ" or another form, update consistently.

After review, send the marked-up sheet back. I'll regenerate `pa.lproj/Localizable.strings` from the corrected entries.

## Drug-info translations

For each entry in `Resources/drug_info_pa.json`, also confirm:

- `whatItDoes` reads at a Grade-6 level (no clinical jargon).
- `howToTake` matches what a BC pharmacist would say.
- `commonSideEffects` and `seriousSideEffects` use words an elderly speaker would recognise (e.g., "ਪੇਟ ਖ਼ਰਾਬ" vs "gastric distress").
- `foodGuide.caution` / `avoid` flags real concerns; warfarin's leafy-green caution is the most important one to verify.
- Drug names in `commonNames` stay in Latin script — those are how the pharmacy bottle is labelled.

## Strings I deliberately left in English

- `[NOTIF-DEBUG]` / `[VOICE-DEBUG]` / `[DRUG-DEBUG]` console log lines — developer surface.
- DEBUG pill button labels ("Test notification (30s)", "Schedule real dose (2 min)", "Clear drug cache") — never shipped to users.
- Source URL hostnames ("dailymed.nlm.nih.gov") — they're URLs, not user copy.
- Drug brand names ("Metformin", "Lisinopril", etc.) inside `commonNames` — they match the English pharmacy bottle.
